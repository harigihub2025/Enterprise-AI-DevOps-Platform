About Enterprise AI DevOps Platform

Enterprise AI DevOps Platform is a modern cloud automation application developed using Spring Boot and MySQL. It demonstrates enterprise application development, REST API integration, Infrastructure as Code, Continuous Integration & Continuous Deployment, Containerization and Cloud deployment.

🎯 Project Goal

Build an enterprise-ready DevOps platform capable of managing projects using modern cloud technologies.

⚡ Key Features

CRUD Operations, REST API, Validation, Exception Handling, Search, Responsive UI.

☁ Cloud Ready

Designed for AWS deployment with Docker, Kubernetes, Jenkins and Terraform.
